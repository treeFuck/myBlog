页面滚动到指定位置：

1.  元素的 scrollIntoView() 、scrollIntoViewIfNeeded() 方法
2.  为元素设置焦点
```js
let str = '';
for (let i = 0; i < 300; i++) {
	str += `<button>${i+1}</button>`
}
document.body.innerHTML = str;

let p = document.body.lastElementChild;

setTimeout(()=>{
	p.focus()
	console.log(document.activeElement)
}, 1000)
```
