16. 编程题2：

    ```js
    实现函数接受任意二叉树，求二叉树所有根到叶子节点路径上所有节点，组成的数字之和
    二叉树每个节点的value范围是1-9
    例如：
          1
       2    3 
     4   5
    从根到叶子共3条：1->2->4, 1->2->5, 1->3
    构成的数字为124，125，13，求和124 + 125 + 13 = 262即为所求
    
    let fun = (p, val)=>{ 
        if (!p) return 0;
        
        if (!p.left && !p.right) {
            // 叶子节点
            return val * 10 + p.val
        }
        
        // 非叶子节点
        let left = fun(p.left, val * 10 + p.val);
        let right = fun(p.right, val * 10 + p.val);
        
        // 求值合并
        return left + right;
    }
    ```
