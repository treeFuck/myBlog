## 一、shell 中特殊且重要的变量

### 1.1 特殊位置参数变量

从命令行、函数、脚本执行处等传递参数时，你想接住这些参数，就需要使用特殊位置参数变量。

介绍一下常用的特殊位置参数变量：

1. ${0} → 当前执行的 shell 脚本的文件名
2. ${n} → 当前执行 shell 脚本的第 n 个参数值
3. ${#} → 当前执行 shell 脚本的参数总数
4. ${*} → 获取当前执行 shell 脚本的全部参数
5. ${@} → 获取当前执行 shell 脚本的全部参数

${\*} 和 ${@} 的区别在于，如果给二者加上双引号：

	+ “${\*}”  →  “${1} ${2} ... ${n}”
	+ “${@}” → “${1}”  “${2}” ... “${n}”
